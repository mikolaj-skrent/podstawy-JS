Tabela wybranych metod i właściwości języka JavaScript:


| Grupa | Metoda / Właściwość | Opis |
| :--- | :--- | :--- |
| **DOM (Elementy)** | document.getElementById("id") | Zwraca referencję do elementu o podanym identyfikatorze |
| **DOM (Elementy)** | document.createElement("tag") | Tworzy nowy element HTML określonego typu (np. "li", "p") |
| **DOM (Elementy)** | element.appendChild(węzeł) | Dodaje nowy element jako ostatnie dziecko danego elementu |
| **DOM (Treść i Styl)** | element.value | Pobiera lub ustawia wartość pola formularza (np. input, select) |
| **DOM (Treść i Styl)** | element.innerHTML | Pobiera lub ustawia treść HTML wewnątrz elementu |
| **DOM (Treść i Styl)** | element.style.cecha | Modyfikuje styl CSS (np. .color, .backgroundColor, .fontSize) |
| **Konwersja** | parseInt("tekst") | Konwertuje ciąg znaków na liczbę całkowitą |
| **Konwersja** | parseFloat("tekst") | Konwertuje ciąg znaków na liczbę zmiennoprzecinkową |
| **Konwersja** | isNaN(wartość) | Sprawdza, czy podana wartość nie jest liczbą |
| **Matematyka** | Math.round(liczba) | Zaokrągla liczbę do najbliższej liczby całkowitej |
| **Matematyka** | Math.floor(liczba) | Zaokrągla liczbę w dół do najbliższej liczby całkowitej |
| **Matematyka** | Math.ceil(liczba) | Zaokrągla liczbę w górę do najbliższej liczby całkowitej |
| **Tekst (String)** | string.length | Zwraca liczbę znaków w ciągu tekstowym |
| **Tekst (String)** | string.toLowerCase() | Zamienia wszystkie litery w ciągu na małe |
| **Tekst (String)** | string.toUpperCase() | Zamienia wszystkie litery w ciągu na wielkie |
| **Tablice (Array)** | array.push(element) | Dodaje nowy element na końcu tablicy |
| **Tablice (Array)** | array.length | Zwraca liczbę elementów znajdujących się w tablicy |
